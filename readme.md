# Installation

git clone pypl-stock-monitor

## Variables de ambiente
Renombrar el archivo .env.example a .env
Configurar los valores con los siguientes pasos

## Crear un bot de Telegram
Buscar en Telegrame el bot *@BotFather*
Crear un nuevo bot con el comando /newbot
Seguir las instrucciones y obtener el hash del bot
Configurar el hash en el archivo .env

## Crear cuenta (gratuita) en Alpa Vantage
Entrar a https://www.alphavantage.co/support/#api-key
Registrarse y obtener el hash del api
Configurar el hash en el archivo .env
